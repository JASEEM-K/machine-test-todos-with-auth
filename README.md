## Example .env file

PORT=3000
MONGO_URI=mongodb://localhost:27017/machinetest-todo-with-auth
JWT_SECRET=your-jwt-secret
FRONTEND_URL=http://localhost:5173

## Build commands

#### Backend

```
cd backend
npm i
npm run build
npm run start

npm run dev // for starting on development mode
```

#### Frontend

```
cd frontend
npm i
npm run build
npm run start

npm run dev // for starting on development mode
```
